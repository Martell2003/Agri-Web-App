Upload CSV File

Upload a CSV file to bulk import price data. The CSV should contain the following columns:

  • product_name: The name of the product (must match an existing product in the database)
  • market_name: The name of the market (must match an existing market in the database)
  • price: The price value (e.g., 100.50)
  • timestamp: The date of the price (e.g., 2025-04-15)

Example CSV:

product_name,market_name,price,timestamp
Maize,Nairobi Market,50.00,2025-04-15
Rice,Mombasa Market,75.00,2025-04-15
